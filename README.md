## Environment
* Install Java jdk 1.8+
* Install Gradle 5+
* Add the main directory to the classpath in .bashrc
** export CLASSPATH=".:./build/classes/java/main:./lib/*"
** source .bashrc
* Untar java bundle or clone the repo
* In the bundle build the project
** gradle assemble
